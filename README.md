# 京东代下单系统

一套完整的京东代预约下单系统，支持**京东家政**和**京东洗衣服**两种业务类型，包含客户H5预约页面和后台管理系统。

## 访问地址

| 用途 | 地址 |
|------|------|
| 客户预约页面 | `http://你的域名/{卡密}` 例如：`http://域名/TJR93VDP9Q3Q9984` |
| 后台管理 | `http://你的域名/admin` |
| 默认账号 | admin / admin123 |

## 核心功能

- ✅ H5预约页面（对标京东家政同行）
- ✅ **双业务类型**：京东家政 / 京东洗衣服
- ✅ 卡密生成与管理（按业务类型区分）
- ✅ **卡密作废功能**：支持将未使用的卡密标记为"已失效"，阻止卡密被使用（不可逆）
- ✅ **订单导出功能**：家政/洗衣订单支持按筛选条件导出 CSV 文件（Excel 可直接打开）
- ✅ 预约时间规则配置（按城市/地区/全局设置锁定天数）
- ✅ 订单管理（按业务类型区分）
- ✅ 京东账号管理（Cookie池）
- ✅ Puppeteer自动下单引擎
- ✅ 数据看板
- ✅ 地区管理（省市区街道四级联动）
- ✅ **鲸蚁生活洗护API对接**（洗衣服务自动下单、状态回调、物流查询）
- ✅ **横向滑动日历时间选择器（仿美团/京东风格）**
- ✅ **多备选时间预约（可配置1-3个备选时间）**
- ✅ **订单备注功能（快速编辑+搜索支持）**
- ✅ **备注图片上传**：家政订单备注支持粘贴/拖拽/点击上传图片，列表显示图片预览图标
- ✅ **图形验证码登录**：后台登录页接入图形验证码（SVG形式），防止暴力破解
- ✅ **子账号权限系统**：支持子账号按模块授权，登录后自动隐藏无权限菜单，路由守卫拦截越权访问
- ✅ **子账号权限实时刷新**：每次路由跳转自动同步数据库最新权限，账号被禁用立即跳转登录页
- ✅ **订单号截断显示**：列表中订单号截断为末尾10位，完整号码通过 tooltip 悬浮查看
- ✅ **订单时间格式统一**：所有时间字段统一转北京时间显示（YYYY-MM-DD HH:mm格式）

## 子账号权限系统

后台管理支持三种角色：

| 角色 | 说明 |
|------|------|
| `super` | 超级管理员，拥有所有权限，可管理子账号 |
| `admin` | 普通管理员，拥有所有功能权限 |
| `sub` | 子账号，仅拥有被授权的模块权限 |

可授权的模块：数据看板、家政订单、家政卡密、京东账号、时间规则、洗衣订单、洗衣卡密、洗衣时间规则、洗衣设置、地区管理、系统设置、子账号管理

**子账号管理入口**：后台菜单 → 子账号管理（仅 super/admin 角色可见）

**实时权限刷新**：每次路由跳转时，前端自动调用 `GET /admin/api/auth/me` 接口从数据库获取最新权限，若账号被禁用则立即跳转登录页，无需用户手动刷新。

## 鲸蚁洗衣API对接

洗衣服务对接**鲸蚁生活洗护**平台API，支持自动下单、状态回调和物流查询。

### 配置方式

1. 登录后台管理 → 系统设置 → 洗衣API设置
2. 填写以下配置项：
   - `洗衣API AppID` / `AppSecret`：鲸蚁开放平台认证信息
   - `工厂代码`：指定洗衣工厂
3. 开启「洗衣自动下单」开关

### Webhook 回调

鲸蚁系统会主动推送订单状态变更到以下地址（无需鉴权）：

```
POST /api/laundry/callback
```

| 状态码 | 含义 | 处理逻辑 |
|--------|------|----------|
| 1 | 已分配 | 写入取件快递单号（按前缀识别顺丰/京东/其他）、工厂信息，订单状态→2 |
| 2 | 已取件 | 更新洗衣状态为「已取件」 |
| 3 | 已入厂 | 更新洗衣状态为「已入厂」 |
| 4 | 预检中 | 保存衣物图片（优先用 images_v2，兼容 images） |
| 5 | 已回寄 | 写入回寄快递单号，更新洗衣状态为「已回寄」 |
| 6 | 已送达 | 订单状态→6 |
| 10 | 完成 | 订单状态→6 |
| 11 | 质检中 | 更新洗衣状态为「质检中」 |
| -1 | 已取消 | 订单状态→4，洗衣状态→「已取消」 |

## 卡密作废说明

管理员可在「卡密管理」→「全部卡密」页面对未使用的卡密执行**作废**操作：

- 作废后，该卡密的 `status` 变为 `2（已失效）`，用户端验证时将提示"卡密已失效"
- **不可逆操作**：作废后无法恢复为未使用状态，也无法删除（需删除时请直接使用删除功能）
- 已使用（`status=1`）的卡密不允许作废

## 订单导出说明

在「家政订单管理」或「洗衣订单管理」页面，使用筛选条件后点击**「导出CSV」**按钮：

- 导出范围：匹配当前所有筛选条件的**全部订单**（不受分页限制）
- 文件格式：CSV（UTF-8 with BOM，可用 Excel 直接打开，中文不乱码）
- 家政订单导出字段：订单号、姓名、手机号、省市区街道、详细地址、预约日期、预约时间、服务类型、服务时长、卡密、状态、京东订单号、备注、创建时间
- 洗衣订单导出字段：订单号、姓名、手机号、省市区、详细地址、预约日期、取件时间段、商品名称、卡密、鲸蚁订单号、洗衣状态、取件快递单号、状态、备注、创建时间
