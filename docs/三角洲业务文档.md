# 三角洲哈夫币业务文档

## 业务概述

三角洲哈夫币（`sjz`）是系统第4个业务模块，提供三角洲游戏哈夫币充值服务。

- 前端路由：`/sjz/:code`
- 前端目录：`frontend-sjz/`
- 后端 API 前缀：`/api/sjz/`
- 订单号格式：`SJZ20260411XXXXXX`

---

## 业务流程

1. 客户打开 `/sjz/{卡密}`
2. 系统验证卡密有效性，返回套餐配置（哈夫币数量、需要填写的字段等）
3. 客户填写：手机号（必填）+ 游戏昵称 / 保险格数 / 是否成年 / 仓库截图（按套餐配置，可选）
4. 客户提交 → 后端创建订单 → 若企业微信已开启：生成专属联系方式（二维码 or 跳转链接）
5. 页面跳转到成功页，展示企业微信入口，引导客户添加
6. 客户添加好友 → 企业微信回调 → 自动备注 / 发欢迎语 / 可选自动建群
7. 后台管理员在"三角洲订单"Tab 查看、手动安排服务

## H5 API 接口

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/sjz/verify-card/:code` | 验证卡密有效性 |
| GET | `/api/sjz/ip-city` | 获取IP归属城市 |
| POST | `/api/sjz/upload` | 上传截图（最多5张，各10MB） |
| POST | `/api/sjz/orders` | 提交三角洲订单 |
| GET | `/api/sjz/qywx-callback` | 企业微信回调验证 |
| POST | `/api/sjz/qywx-callback` | 企业微信回调事件 |

## 企业微信 API 说明

系统使用企业微信客户联系相关接口：

| 接口 | 说明 |
|------|------|
| `POST /externalcontact/add_contact_way` | 生成专属"联系我"方式 |
| `POST /externalcontact/remark` | 自动备注客户 |
| `POST /externalcontact/send_welcome_msg` | 发送欢迎语（首次添加触发） |
| `POST /appchat/create` | 创建客户群 |
| 回调事件 `add_external_contact` | 客户添加好友事件 |

### 回调事件 state 格式

```
SJZ_{订单号}_{员工ID}
```

例如：`SJZ_SJZ20260411123456_员工userid`

## 企业微信申请教程

### 步骤一：注册企业微信

访问 https://work.weixin.qq.com/ 注册企业

### 步骤二：开启客户联系 API

1. 登录企业微信管理后台
2. 进入：客户联系 → API → 开启
3. 获取**客户联系 Secret**（填入后台"企业微信配置"中的 `qywx_secret`）

### 步骤三：配置接收事件服务器

1. 在企业微信管理后台：客户联系 → API → 接收事件服务器
2. 填写 URL：`{你的域名}/api/sjz/qywx-callback`
3. 填写 Token 和 EncodingAESKey（与后台配置保持一致）
4. 点击保存，系统自动完成验证

### 步骤四：配置企业可信 IP

在企业微信管理后台：企业信息 → 功能 → IP白名单，填入服务器 IP

### 费用说明

全部接口**免费**，无需付费。
