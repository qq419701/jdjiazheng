# 宝塔面板部署教程

部署完成后通过 Nginx 统一反向代理到端口5500，所有入口：

| 用途 | 地址 |
|------|------|
| 家政预约页 | `http://你的域名/jz/{卡密}` |
| 洗衣预约页 | `http://你的域名/xi/{卡密}` |
| 充值预约页 | `http://你的域名/cz/{卡密}` |
| 后台管理 | `http://你的域名/admin` |

---

## 第一步：安装必要软件

宝塔面板 → 软件商店，安装：
1. **Node.js版本管理器** → 安装 Node.js **18.x**
2. **MySQL 5.7**（记住root密码）
3. **PM2管理器**
4. **Nginx**

---

## 第二步：创建数据库

宝塔 → 数据库 → 添加数据库：
- 数据库名：`jdjiazheng`
- 用户名：`jdjiazheng`
- 密码：自定义（记住）

---

## 第三步：上传代码

```bash
cd /www/wwwroot
git clone https://github.com/qq419701/jdjiazheng.git
```

---

## 第四步：配置环境变量

```bash
cd /www/wwwroot/jdjiazheng/backend
cp .env.example .env
```

编辑 `.env`：
```env
DB_HOST=127.0.0.1
DB_PORT=3306
DB_NAME=jdjiazheng
DB_USER=jdjiazheng
DB_PASS=你的数据库密码
JWT_SECRET=随机字符串32位以上
PORT=5500
NODE_ENV=production
```

---

## 第五步：安装依赖 + 初始化数据库

```bash
cd /www/wwwroot/jdjiazheng/backend
npm install
node init/initDB.js
```

看到 `✅ 数据库初始化完成` 即成功。默认管理员：**admin / admin123**

---

## 第六步：PM2启动后端

宝塔 → PM2管理器 → 添加项目：
- 启动文件：`/www/wwwroot/jdjiazheng/backend/app.js`
- 运行目录：`/www/wwwroot/jdjiazheng/backend`
- 项目名称：`jdjiazheng`

---

## 第七步：Nginx反向代理

宝塔 → 网站 → 添加站点（填写域名）→ 设置 → 反向代理：
- 目标URL：`http://127.0.0.1:5500`

---

## 第八步：申请SSL（推荐）

宝塔 → 网站 → SSL → Let's Encrypt → 申请 → 开启强制HTTPS

---

## 验证

1. 访问 `http://你的域名/admin`
2. 账号 `admin`，密码 `admin123`
3. 登录后到卡密工作台 → 生成卡密 → 访问对应链接 → 部署成功 🎉

## 常见问题

**Q：访问显示404？**
A：检查PM2是否运行；检查Nginx反向代理目标是否为 `127.0.0.1:5500`

**Q：数据库连接失败？**
A：检查 `.env` 中 `DB_PASS` 是否正确；确认MySQL服务是否运行

**Q：卡密页面白屏？**
A：需执行前端构建命令（见上方"更新代码"）

**Q：PM2重启后服务未恢复？**
A：在PM2管理器中设置"开机自启"
