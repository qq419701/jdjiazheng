# 京东代下单管理系统 - 技术文档 v3.0

## 一、系统概览

本系统为**三业务合一**的卡密代下单管理平台，支持家政、洗衣、充值三类业务的卡密分发、订单管理和后台运营。

### 三大业务

| 业务 | 前端路径 | 前端目录 | 核心流程 |
|------|----------|----------|----------|
| 🏠 家政 | `/{site}/jz/{卡密}` | `frontend-h5/` | 用户填写预约信息 → 系统Puppeteer自动在京东下单 |
| 🧺 洗衣 | `/{site}/xi/{卡密}` | `frontend-xi/` | 用户填写取/收件地址 → 自动推送鲸蚁生活系统 |
| 💳 充值 | `/{site}/cz/{卡密}` | `frontend-cz/` | 用户填写充值账号 → 人工充值 → 管理员确认完成 |

## 四、后台管理功能模块

| 模块 | 路由 | 权限Key | 说明 |
|------|------|---------|------|
| 数据看板 | `/admin/dashboard` | `dashboard` | 订单统计趋势图 |
| 订单中心 | `/admin/order-center` | `order_center` | 家政/洗衣/充值三业务订单统一管理 |
| 卡密工作台 | `/admin/card-workbench` | `card_workbench` | 生成/查询/批次管理（三业务统一） |
| 套餐管理 | `/admin/template-manager` | `template_manager` | 管理卡密套餐模板 |
| 业务设置 | `/admin/business-settings` | `business_settings` | 家政/洗衣/充值/SUP/通用设置（含京东账号管理、时间规则管理） |
| SUP日志 | `/admin/sup-logs` | `sup_logs` | 阿奇所SUP接口调用日志 |
| 地区管理 | `/admin/regions` | `regions` | 地区设置 |
| 子账号管理 | `/admin/sub-accounts` | `sub_accounts` | 子账号权限分配 |

> **注意：京东账号管理是家政设置的内嵌功能（在业务设置 → 家政设置 Tab 内），不是独立的系统模块，没有独立的侧边栏入口。**

## 五、子账号权限体系

子账号（role=sub）可被分配以下权限模块：

| 权限Key | 说明 |
|---------|------|
| `dashboard` | 数据看板 |
| `card_workbench` | 卡密工作台 |
| `template_manager` | 套餐管理 |
| `order_center` | 订单中心 |
| `business_settings` | 业务设置（含时间规则、京东账号管理） |
| `regions` | 地区管理 |
| `sub_accounts` | 子账号管理 |
| `sup_logs` | SUP日志 |

> admin / super 角色拥有全部权限，无需单独分配。

## 六、数据库核心表

### cards 卡密表
`business_type` 区分业务：`jiazheng` / `xiyifu` / `topup`

### orders 订单表
三业务共用一张表，通过 `business_type` 字段区分。

### settings 系统设置表
键值对存储，详细key列表见initDB.js。

### products 套餐（商品）表
管理卡密套餐模板，卡密生成时关联 `product_id`。

### admins 管理员表
支持 `super` / `admin` / `sub` 三种角色。

---

## 七、时间规则引擎

按城市匹配预约时间，优先级：**city精确 > tier地区 > global全局**

时间段支持：`08:00` ~ `18:00`（可在业务设置中按规则配置）

---

## 八、Puppeteer自动下单（家政）

通过京东账号池自动在京东平台下单家政服务。
- 支持账号池轮询（`is_busy` 状态控制并发）
- 失败自动重试3次
- 支持Cookie注入登录

---

## 九、鲸蚁生活API对接（洗衣）

洗衣订单提交后自动推送到鲸蚁生活洗护系统，支持：
- 自动获取Token / 使用固定 out-token
- 物流轨迹查询
- 状态回调接收

配置路径：业务设置 → 洗衣设置 → 鲸蚁API配置

---

## 十、阿奇所SUP系统对接

作为货源方接入阿奇所开放平台，支持卡密销售和退款。

接口路径：`/agisoAcprSupplierApi/*`

配置路径：业务设置 → 奇所SUP设置
